### Utility that calculates differences in text files

### Usage
gendiff (-h|--help)

gendiff (-v|--version)

gendiff [--format ]

### Format options:

stylish

plain

json

### Different formats are output by command:
```

$ ./gendiff <file 1> <file 2> --format <option> 

```

### Requirements:

PHP 8.0

Composer 2.*

### Installation:
```

$ git clone git@github.com:alexx134121/php-project-48.git

$ cd php-project-48

$ make install

```
